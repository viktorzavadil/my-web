# viktorzavadil.cz

Osobní web a blog.

## Tech Stack

- **Astro 5** - static site generator
- **Tailwind CSS 4** + **DaisyUI 5** - styling
- **MDX** - blog content

## Příkazy

```bash
npm run dev      # dev server
npm run build    # produkční build
npm run preview  # náhled buildu
```

## Struktura

```
src/
├── components/    # Astro komponenty
├── content/blog/  # MDX články
├── layouts/       # BaseLayout
├── pages/         # routing
└── styles/        # global.css
```
